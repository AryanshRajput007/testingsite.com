What is File Processing System?

File Processing System (FPS) is a way of storing, retrieving and manipulating data which is present in various files.

Some Key Features of FPS :-
  1. File Creation and Deletion :- The file system enables users and applications to create new files and delete existing ones.
  2. File Organization :- It determines how files are organized on the storage media, such as how data is allocated and stored in blocks or clusters.
  3. File Access and Retrieval :- The file system facilitates reading and writing data from and to files, ensuring that the correct data is retrieved and that updates are appropriately saved.
  4. File Security :- File systems implement access control mechanisms to restrict or grant permissions for reading, writing, and executing files.

Advantages of FPS :-

  • Cost Friendly :- We don't have to buy any software as we can use version which are already installed in our Systems.
  • Easy to use :- File systems require very basic learning and understanding, hence, it can be easily used.
  • High scalability :- One can very easily switch from smaller to larger files as per his needs.

Disadvantages of FPS :-

  • Data redundancy :- Data redundancy in a file processing system occurs when the same data is duplicated or repeated in multiple files. Since data is stored in separate files, it is common to have multiple copies of the same information scattered across various files.
  • Data Inconsistency :- Data inconsistency in a file processing system occurs when the same data exists in multiple files but is not synchronized or updated consistently.
  • Data Isolation :- In an FPS, data is isolated and fragmented across different files, making it difficult to access and share data among multiple applications. This lack of integration hinders data sharing and can result in data duplication, leading to inefficiencies and difficulties in data maintenance.
  • Data Security :- FPS often lack robust security measures compared to modern DBMS. With decentralized data storage, it becomes harder to implement comprehensive security controls, leaving data more vulnerable to unauthorized access or breaches.
